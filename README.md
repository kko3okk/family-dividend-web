# 家的冒險 · Dividend Quest

把「十四年股息累積計畫」變成寶可夢式冒險的家庭進度板。
夫妻兩人各自記錄買進，年股息就是攻擊力，里程碑是道館徽章，終點是魔王城。

**網址**：https://kko3okk.github.io/family-dividend-web/

---

## 這是什麼

不是記帳軟體，也不是看盤工具。它只做一件事：**讓一個需要走十幾年的計畫，每天打開都還想繼續**。

刻意不顯示的東西：即時股價漲跌、未實現損益、報酬率。
因為這個計畫的勝負在現金流累積，不在每天賺賠——看了只會誘發不必要的操作。

## 遊戲系統 ↔ 現實對照

| 遊戲 | 現實 |
|------|------|
| ⚔ 攻擊力 ATK | 年股息（股數 × 每股股利，全年合計） |
| 對手 HP | 里程碑目標金額 |
| 已造成傷害 | 目前年股息 |
| 一次攻擊 | 一筆買進（傷害 = 股數 × 每股股利） |
| 等級 Lv | 每 10,000 元年股息升一級 |
| 道館徽章 | 三個里程碑 |

### 三座道館

| 徽章 | 年股息目標 | 意義 |
|------|-----------|------|
| 旅遊道館 | 150,000 | 每年出國由股息買單 |
| 房貸道館 | 610,000 | 房貸（1,000 萬 / 30 年）＋旅遊全額 cover |
| 魔王城 | 1,000,000 | 退休線 · 年被動收入百萬 |

## 功能

- **戰鬥畫面**：GBA 風格 HP／EXP 資訊框，自動播報最近一次攻擊
- **雙人切換**：勇者（Chang）與魔法使（老婆）各自的持股、進度、戰術指引
- **成本殖利率**：依交易紀錄計算加權平均成本與 YOC
- **除權息倒數**：支援年配／半年配／季配；腳本自動抓證交所公告，未確認者標「預估」
- **戰利品入帳**：發放日自動產生待確認股息（已估算補充保費與匯費），核對存摺後一鍵確認
- **年度戰績**：每年 1/1 自動快照，長條圖比較「預估年股息 vs 實際入帳」，附年度檢核清單
- **買點指引**：南寶依觸發價（股利 ÷ 5.5%／6%／6.5%）自動判定狀態；老婆組合依除權息季節切換窗口提示
- **音效**：程式生成的原創環境音與戰鬥音效；放入 `bgm.m4a` 可換成自己的音樂
- **PWA**：可加入主畫面全螢幕使用，含下拉重整與自動版本更新

## 架構

```
family-dividend-web   (公開)   網站本體，GitHub Pages 部署
  index.html                   整個應用（單檔：樣式 + 邏輯）
  version.json                 版本號，供自動更新比對
  px/*.png                     像素素材（原創，程式繪製）
  manifest.json, icon-*.png    PWA 設定與圖示

family-dividend-data  (私有)   資料庫
  data.json                    持股、股利、交易、入帳、徽章、檢核
  backups/YYYY-MM-DD.json      每日自動備份
```

**沒有後端、沒有資料庫。** GitHub API 即是儲存層：
網頁讀寫私有 repo 的 `data.json`，Token 只存在各自手機的瀏覽器裡。
公開的只有網站程式碼，持股資料在私有 repo，路人打開只會看到「請設定 Token」。

### 資料保存原則

- 雲端（換手機不遺失）：交易紀錄、股息入帳、年度快照、檢核清單、徽章取得紀錄
- 本機（刻意）：GitHub Token（憑證）、預設顯示的角色（個人偏好）
- 三重保護：當前檔案 + 每日備份 + GitHub commit 歷史（每筆交易都是一個 commit）

## 腳本

| 檔案 | 用途 | 排程 |
|------|------|------|
| `dividend_alert.py` | 到價推播、現價回寫、除權息公告抓取、待確認入帳、年度快照 | 交易日 09:05–13:30 每 15 分 |
| `backup_data.py` | data.json 每日快照備份 | 每日 22:00 |
| `deploy.sh` | 產生新版本號、推送、觸發 Pages 重建 | 手動 |

```bash
# 到價提醒 + 現價回寫
5-59/15 9-13 * * 1-5 GH_DATA_TOKEN=xxx python3 /path/dividend_alert.py
# 每日備份
0 22 * * * GH_DATA_TOKEN=xxx python3 /path/backup_data.py
# 部署
GH_TOKEN=xxx ./deploy.sh
```

## 每年三月要做的事

1. 各檔股利公告後，更新 `data.json` 的 `div`（觸發價、年股息、徽章進度全部自動重算）
2. 檢查熔斷條件（見下）
3. 結算去年主動獲利，執行 40% 搬運
4. 完成年度戰績卡的檢核清單

### 熔斷條件（觸發即停買該檔）

- 南寶：年度配息 < 15 元 ／ EPS 連兩年衰退 ／ 半導體封裝膠業務停滯
- 崑鼎：配息停止成長 ／ 焚化爐合約流標 ／ EPS 連兩年衰退
- 單檔市值不得超過總資產 15%
